# Technical Projects

## Digital Forensics Investigation

This project involved analyzing seized digital devices in a simulated criminal investigation involving suspected drug trafficking.

**Tools Used**

- FTK Imager
- Autopsy

**Project Tasks**

- Created a forensic disk image using FTK Imager
- Verified data integrity using hash values
- Analyzed the disk image using Autopsy
- Recovered deleted files, text messages, and images
- Investigated email communications and metadata
- Identified location data and communication patterns

**Outcome**

The forensic investigation revealed communication between suspects regarding drug transactions, including emails, messages, and location data connected to meetings in Frostburg, Maryland.

## DDoS / SYN Flood Attack Simulation Lab

This project demonstrated how denial-of-service attacks impact network systems using a controlled virtual lab environment.

**Environment**

- Kali Linux
- Windows Virtual Machine
- VMware Workstation

**Tools Used**

- Metasploit
- Python
- Scapy

**Project Activities**

- Configured a virtual network lab
- Simulated SYN flood attacks using Metasploit
- Created a Python script using Scapy to generate TCP SYN packets
- Observed network traffic and system behavior during the attack
- Studied detection and mitigation strategies

**Outcome**

The project demonstrated how easily SYN flooding attacks can overwhelm network resources and highlighted the importance of monitoring, firewall protections, and defensive security strategies.
